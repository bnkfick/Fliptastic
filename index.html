ANGULAR
Angular (formerly Angular.js): Front-end web app framework; runs your JavaScript code in the user’s browser, allowing your application UI to be dynamic.

What is Angular?

Angular is a JavaScript Framework which allows you to create reactive Single-Page Applications (SPA’s).

What is a Component?

Components are building blocks of Angular applications.
It is possible to nest components within other components and also reuse them when required.

What is a component composed of?

Components include the template, the class, the @component decorator and metadata(template, style, selector) telling angular that the class is a component

What is a Decorator?

– a feature of Typescript

A function that provides information about the class attached to it i.e. for component we use the component decorator @Component

What is a selector?

-selector a custom html tag representing the component

What is Databinding?

Databinding is common communication between your types of code - between your component, your business logic and the template because you fetched something from a server or finished some calculation which you want to display to the user.

What is Interpolation?

A class property within double curly braces is known as interpolation. This is the simplest way to bind data from a class to the template. {{aproperty}} Only works with string values.

Event Binding

To respond to user events such as mouseclicks or keyboard events we need the data to flow from the template to the class. Event binding enables this.
